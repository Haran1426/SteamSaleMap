Architecture

데이터 연동 구조

Steam 현재가

Netlify Function이 Steam Store 데이터를 호출해 현재가, 정가, 할인율, 할인 종료일을 병합합니다.

가격 이력

ITAD API 키가 있으면 역대 최저가와 최근 가격 이력을 붙여 구매 판단 근거를 늘립니다.

장애 대응

실시간 API 실패 시 `games.json` 샘플 데이터를 사용해 프론트엔드 데모가 계속 동작합니다.

운영 관점

동기화 상태, 데이터 소스, CSV 내보내기를 제공해 이커머스/ERP성 데이터 관리 화면처럼 설명할 수 있습니다.

실제 구매 전 Steam 상점 가격과 지역 제한을 최종 확인하세요. 일부 게임은 연령 등급이 있을 수 있습니다.
