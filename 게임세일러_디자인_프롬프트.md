# 게임세일러 디자인 리팩토링 프롬프트

## 역할

너는 시각적으로 강렬하고 기억에 남는 웹 대시보드를 만드는 프론트엔드 디자이너다.
기능 코드(`main.js`, `netlify/functions/`, `games.json`)는 **절대 수정하지 마라.**
오직 `index.html`의 구조와 `style.css`만 수정한다.

---

## 프로젝트 맥락

- **사이트명**: 게임세일러 (GameSailor)
- **목적**: Steam 게임 할인 정보를 수집·분석해 구매 추천 점수로 정렬하는 포트폴리오 대시보드
- **스택**: HTML + Vanilla CSS + Vanilla JS, Netlify 정적 배포
- **기존 JS가 의존하는 DOM 선택자**: 아래 목록의 id/class는 반드시 유지해야 한다

```
#game-grid          — 게임 카드 컨테이너
#search-input       — 검색 인풋
#sort-select        — 정렬 셀렉트
#filter-genre       — 장르 필터
#filter-tag         — 태그 필터
#export-csv         — CSV 내보내기 버튼
#sync-status        — API 동기화 상태 표시
.game-card          — 개별 게임 카드
.game-title         — 카드 내 게임 제목
.game-discount      — 할인율 뱃지
.game-price-now     — 현재가
.game-price-original — 정가 (취소선)
.game-score         — 추천 점수
.game-lowest-diff   — 역대 최저가 대비 차이
.game-end-date      — 할인 종료일
```

## 디자인 방향

**컨셉: "데이터 브리지" — 해군 함교(bridge)의 계기판**

Steam의 어두운 네이비를 기반으로, 마치 군함 계기판처럼 조밀하고 신뢰감 있는 정보 밀도를 표현한다.
단순한 "다크 모드 카드 리스트"가 아니라, 데이터를 항해하듯 탐색하는 느낌을 준다.

### 색상 토큰 (CSS 변수로 정의)

```css
--bg-depth-0: #0a1220;   /* 최심부 배경 */
--bg-depth-1: #111d2e;   /* 사이드바/헤더 */
--bg-depth-2: #172032;   /* 메인 배경 */
--bg-depth-3: #1c2a40;   /* 카드 배경 */
--bg-hover:   #243552;   /* 호버 상태 */

--accent-blue:  #4a9edd;  /* 주요 강조, 링크 */
--accent-green: #a4d007;  /* 할인율, 추천 뱃지 */
--bg-green:     #2a3d0f;  /* 할인율 뱃지 배경 */
--accent-amber: #f0a500;  /* 경고/주의 강조 */
--accent-red:   #e05c5c;  /* 역대 최저가와 차이 큰 경우 */

--text-primary:   #c7d5e0;  /* 본문 */
--text-secondary: #7a9ab0;  /* 보조 */
--text-muted:     #455a6e;  /* 힌트, 비활성 */

--border-default: rgba(100, 160, 220, 0.12);
--border-hover:   rgba(100, 160, 220, 0.28);
```

### 타이포그래피

- 헤더/로고: `'Orbitron'` (Google Fonts) — 테크/항법 계기판 느낌
- 숫자/가격: `'JetBrains Mono'` (Google Fonts) — 데이터 가독성
- 본문/레이블: `'Noto Sans KR'` (Google Fonts) — 한글 지원

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Orbitron:wght@600;800&family=JetBrains+Mono:wght@400;600&family=Noto+Sans+KR:wght@400;500&display=swap" rel="stylesheet">
```

## 레이아웃 명세

### 전체 구조

```
┌─────────────────────────────────────────────────┐
│  HEADER: 로고 + 동기화 상태(#sync-status) + 우측 정렬  │
├──────────┬──────────────────────────────────────┤
│ SIDEBAR  │  MAIN CONTENT                        │
│ 필터 패널 │  검색바 + 정렬 + 게임 그리드(#game-grid) │
│ (240px)  │                                      │
└──────────┴──────────────────────────────────────┘
```

- 전체 레이아웃: `display: grid; grid-template-columns: 240px 1fr;`
- 사이드바: sticky, 높이 100vh, `--bg-depth-1` 배경
- 메인: `padding: 24px 32px`
- 반응형: 768px 이하에서 사이드바 숨김 → 상단 드로어로 전환

### 헤더

- 배경: `--bg-depth-1`
- 하단 border: `1px solid var(--border-default)`
- 로고 텍스트: `Orbitron 800`, `--accent-blue` 색
- 로고 앞 아이콘: ⚓ 또는 🧭 이모지 or SVG 닻 아이콘
- `#sync-status`: 우측 정렬, 초록 점(●) + "Live" 텍스트, 애니메이션 pulse

```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50%       { opacity: 0.3; }
}
.sync-dot { animation: pulse 2s infinite; color: var(--accent-green); }
```

### 사이드바 필터 패널

- 섹션 구분선: `1px solid var(--border-default)`
- 레이블: `Noto Sans KR 500`, 11px, `--text-muted`, uppercase letter-spacing
- `select`, `input[type=text]`: 배경 `--bg-depth-0`, border `--border-default`, 색 `--text-primary`, focus 시 border `--accent-blue`
- `#export-csv` 버튼: 사이드바 하단 고정, `--accent-blue` border, 호버 시 배경 채움

### 검색 + 정렬 바

- `#search-input`: 전체 너비, 높이 44px, 좌측 검색 아이콘(SVG)
- `#sort-select`: 우측 정렬, 너비 180px
- 바 하단: `8px` 간격 후 게임 그리드 시작

### 게임 카드 (`#game-grid`, `.game-card`)

그리드: `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr)); gap: 16px;`

카드 구조:
```
┌─────────────────────────────┐
│ [썸네일 영역 - 16:9 비율]    │
│                    [-40%]   │ ← .game-discount 우상단 뱃지
├─────────────────────────────┤
│ .game-title (2줄 clamp)     │
│                             │
│ ★★★★☆  [추천점수 바]  .game-score │
├─────────────────────────────┤
│ ₩47,880    ~~₩79,800~~     │
│ .game-price-now  .game-price-original │
│                             │
│ 역최저 -12%  |  D-3         │
│ .game-lowest-diff  .game-end-date │
└─────────────────────────────┘
```

카드 세부:
- 배경: `--bg-depth-3`, border-radius: `10px`, border: `0.5px solid var(--border-default)`
- 호버: `border-color: var(--border-hover)`, `transform: translateY(-2px)`, `transition: 200ms`
- 썸네일: `object-fit: cover`, 없을 경우 `--bg-depth-0` 배경 + 게임 이름 이니셜
- `.game-discount` 뱃지: 우상단 `position: absolute`, 배경 `--bg-green`, 텍스트 `--accent-green`, `font-family: JetBrains Mono`, `font-weight: 600`
- `.game-score` 추천 점수: 숫자 + 얇은 progress bar (`--accent-blue` fill), `JetBrains Mono`
- `.game-price-now`: `JetBrains Mono 600`, `--accent-blue`
- `.game-price-original`: `JetBrains Mono 400`, `--text-muted`, `text-decoration: line-through`
- `.game-lowest-diff`: 최저가보다 낮으면 `--accent-green`, 높으면 `--accent-amber`
- `.game-end-date`: D-3 이하면 `--accent-red` + 깜박임 애니메이션

## 마이크로 인터랙션

1. **카드 로드**: 카드가 순차적으로 페이드+슬라이드업 (`animation-delay: calc(var(--i) * 40ms)`)
2. **할인율 뱃지**: 50% 이상이면 `--accent-amber`로 색 변경
3. **검색 인풋**: 입력 시 실시간 카드 필터 (기존 JS 유지, CSS transition으로 숨김)
4. **정렬 변경**: 카드 순서 바뀔 때 레이아웃 transition `300ms ease`
5. **CSV 버튼**: 클릭 시 짧은 `scale(0.96)` 피드백

## 금지 사항

- 보라색 그라데이션, Inter/Roboto 폰트 사용 금지

## 결과물 형식

`index.html`과 `style.css` 두 파일을 완전한 형태로 출력한다.
기존 HTML의 `<script src="main.js">` 태그는 반드시 유지한다.
Google Fonts `<link>` 태그는 `<head>` 최상단에 추가한다.
